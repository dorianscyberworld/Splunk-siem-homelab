# Lab 002 – Splunk Basics

## Objective
Learn how to navigate Splunk, search logs, and investigate security events.

---

## What I Learned
Splunk allows security analysts to search, filter, and analyze machine-generated data from multiple systems.

This helps analysts investigate suspicious behavior and identify security threats.

---

## Topics Covered
- Splunk navigation
- Search functionality
- Event filtering
- Log analysis
- Security investigation basics

---

## Key Skills Practiced
- Searching all events
- Filtering by source
- Investigating source IP addresses
- Reviewing usernames
- Examining event details

---

## Why This Matters
Splunk is widely used in Security Operations Centers (SOCs) for monitoring environments and investigating suspicious activity.

Examples include:
- Failed login investigations
- Suspicious VPN activity
- Unauthorized access attempts
- Threat hunting
