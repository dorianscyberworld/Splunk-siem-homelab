# Lab 001 – Introduction to SIEM

## Objective
Learn the fundamentals of Security Information and Event Management (SIEM).

---

## What I Learned
SIEM stands for Security Information and Event Management.

SIEM platforms collect logs from multiple systems and centralize them into one location so security teams can monitor, detect, and investigate suspicious activity.

---

## Core Functions
- Log aggregation
- Event correlation
- Threat detection
- Alert generation
- Incident response support

---

## Why This Matters
SIEM tools help organizations improve visibility across their environment and detect security threats faster.

Examples of events monitored:
- Failed login attempts
- Suspicious network activity
- Unauthorized access
- Malware alerts

---

## Career Connection
This lab strengthened my understanding of security monitoring and how SOC analysts use SIEM platforms to investigate threats.
