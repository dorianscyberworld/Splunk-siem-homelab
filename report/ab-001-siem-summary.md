# Lab 001 – SIEM Summary Report

## Objective
Summarize key concepts learned from the Introduction to SIEM lab.

---

## What is SIEM?
SIEM stands for Security Information and Event Management.

A SIEM platform collects and analyzes logs from multiple systems in one centralized location to help security teams monitor and investigate suspicious activity.

---

## Core Functions
- Log aggregation
- Event correlation
- Threat detection
- Alert generation
- Incident support

---

## Why SIEM Matters
SIEM helps organizations improve security visibility and detect threats more efficiently.

Examples include:
- Failed logins
- Suspicious traffic
- Unauthorized access
- Security alerts
